# 🎛 Мастер-Бот — управление 3 каналами из одного места

Один Telegram-бот управляет тремя каналами сразу:

| Канал | Режим | Расписание |
|-------|-------|-----------|
| 📰 Новости | Вручную по кнопке | По запросу |
| 🤖 AI-Постер | Автомат + вручную | 09:00 / 14:00 / 19:00 МСК |
| 🧠 Саморазвитие | Автомат + вручную | 08:00 / 13:00 / 20:00 МСК |

**Как работает:** пишешь боту → он показывает preview поста → ты жмёшь "Опубликовать" или "Перегенерировать".

---

## 🖥 Как выглядит управление

```
/start или /menu  →  главное меню с кнопками

[ 📰 Новость ]  [ 🤖 AI-Пост ]  [ 🧠 Саморазвитие ]
[        📊 Статус каналов        ]

При нажатии на кнопку:
→ Бот генерирует пост (10-30 сек)
→ Показывает preview
→ [ ✅ Опубликовать ]  [ 🔄 Перегенерировать ]  [ ❌ Отмена ]
```

Также работают текстовые команды: `/news` `/post` `/selfdev` `/status`

## 🚀 Запуск: пошаговая инструкция

### Шаг 1 — Создай управляющего бота
1. **@BotFather → /newbot** → придумай имя (например: `My Channel Manager`)
2. Сохрани токен: `7123456789:AAF_...`

### Шаг 2 — Узнай свой Telegram ID
1. Напиши **@userinfobot** в Telegram
2. Он ответит: `Id: 123456789` — это твой ADMIN_ID

### Шаг 3 — Создай трёх ботов для каналов (или используй одного)
Каждый канал нужно связать с ботом-администратором.
Можно использовать **одного и того же** управляющего бота для всех трёх каналов — это проще.

Для каждого канала:
1. Канал → **Настройки → Администраторы → Добавить**
2. Найди своего бота → дай право **"Публикация сообщений"**

### Шаг 4 — Получи Anthropic API Key
1. **console.anthropic.com** → регистрация (дают $5 бесплатно)
2. **API Keys → Create Key** → сохрани `sk-ant-...`

### Шаг 5 — Деплой на Railway

1. Зайди на **railway.app** → Sign in with GitHub
2. **New Project → Deploy from GitHub repo**
3. Подключи свой GitHub и выбери репозиторий с этим кодом
   (сначала залей код на GitHub: создай приватный репо, загрузи все файлы)
4. Railway сам найдёт `Procfile` и поймёт как запустить

### Шаг 6 — Добавь переменные окружения в Railway

В Railway → твой проект → **Variables** → добавь все переменные из `.env.example`:

| Переменная | Значение |
|-----------|---------|
| `BOT_TOKEN` | Токен управляющего бота |
| `ADMIN_ID` | Твой Telegram ID (цифры) |
| `ANTHROPIC_API_KEY` | `sk-ant-...` |
| `NEWS_CHANNEL_ID` | `@канал_новостей` |
| `NEWS_CHANNEL_NAME` | `📰 Мои Новости` |
| `NEWS_NICHE` | `tech` / `ai` / `crypto` / `science` / `business` |
| `NEWS_LANGUAGE` | `russian` / `english` / `both` |
| `POST_CHANNEL_ID` | `@канал_постов` |
| `POST_CHANNEL_NAME` | `🤖 Мой Постер` |
| `POST_TOPIC` | Тема канала (своими словами) |
| `POST_LANGUAGE` | `russian` / `english` / `both` |
| `SELFDEV_CHANNEL_ID` | `@канал_саморазвития` |
| `SELFDEV_CHANNEL_NAME` | `🧠 Саморазвитие` |
| `SELFDEV_LANGUAGE` | `russian` / `english` / `both` |

### Шаг 7 — Запусти и проверь

1. После добавления переменных Railway автоматически перезапустит бот
2. Напиши своему боту `/start`
3. Должно появиться меню с кнопками 🎉

## ⏰ Расписание автопостинга

**AI-Постер** (UTC / МСК):
- `06:00 UTC` = 09:00 МСК
- `11:00 UTC` = 14:00 МСК
- `16:00 UTC` = 19:00 МСК

**Саморазвитие** (UTC / МСК):
- `05:00 UTC` = 08:00 МСК
- `10:00 UTC` = 13:00 МСК
- `17:00 UTC` = 20:00 МСК

Тон постов саморазвития автоматически меняется: утром заряжает, днём — практика, вечером — рефлексия.

## ❓ Частые вопросы

**Бот не отвечает?**
→ Railway → твой сервис → Logs — посмотри ошибки

**"Доступ запрещён" при /start?**
→ Проверь что `ADMIN_ID` = твой настоящий Telegram ID (число без пробелов)

**Бот не публикует в канал?**
→ Убедись что бот добавлен как администратор с правом "Публикация сообщений"

**Railway засыпает?**
→ На бесплатном плане Railway не засыпает для worker-процессов (в отличие от web). Procfile настроен правильно — тип `worker`

**Хочу изменить расписание?**
→ Открой `config.py`, найди `POST_SCHEDULE` и `SELFDEV_SCHEDULE`, измени часы (в UTC)
